## 学生管理系统
### 部署方法

- 下载代码
```
git clone https://github.com/974988176/StudentCMS.git
```

- 安装依赖包
```
composer install
```

- 创建数据库`stucms`.在.env文件中`DB_DATABASE`选项中可更改

- 更改.env里面`DB_USERNAME`等数据库配置

- 执行命令
```cmd
php artisan stucms:run
```
